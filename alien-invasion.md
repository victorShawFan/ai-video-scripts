# 外星入侵题材12秒视频剧本集
## 剧本1
1.  **改编自**：日常通勤遭遇外星突袭母题
2.  **故事原型**：社畜挤在晚高峰地铁车厢里，耳机隔绝了周遭嘈杂，车厢灯光突然毫无规律地频闪，外壁浮现出不属于地球的诡异轮廓，麻木的人群瞬间陷入慌乱。
3.  **12秒改编**：
    - **0-4秒**：地铁车厢暖黄灯光忽明忽暗，穿藏青西装的男人靠在车门打哈欠，耳机线挂在领口，脚边散落半瓶橘子汽水，车厢里飘着廉价洗发水和热干面的混合味道。
    - **4-8秒**：男人余光瞥见窗外掠过银灰色扭曲多面体，玻璃表面浮现粘稠的淡紫色纹路，邻座女生猛地扯下耳机发出短促惊叫，车厢广播戛然而止。
    - **8-12秒**：地铁车门突然弹开，远处站台天花板裂开巨大缝隙，淡蓝色光柱倾泻而下，男人僵在原地，瞳孔骤然收缩。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K超高清，晚高峰地铁车厢暖黄光影忽明忽暗，疲惫社畜靠在车门打哈欠，窗外掠过银灰色扭曲多面体，玻璃浮现淡紫色粘稠纹路，站台天花板裂开缝隙降下淡蓝色光柱，整体氛围压抑惊悚，细节拉满

## 剧本2
1.  **改编自**：独居深夜遭遇外星窥探母题
2.  **故事原型**：独居女生深夜窝在沙发上追剧，客厅只有电视冷光照明，窗台传来细碎刮擦声，她以为是野猫凑近查看，却发现了悬浮的诡异外星物体。
3.  **12秒改编**：
    - **0-4秒**：暖黄落地灯只亮了半边光区，女生裹着珊瑚绒毯子窝在布艺沙发上，手里举着冰奶茶，电视播放着无厘头喜剧，茶几散落半包番茄味薯片和快充数据线。
    - **4-8秒**：女生听见窗台传来细碎声响，按下电视暂停键侧耳倾听，放下奶茶踮脚挪到窗边，手指搭在冰凉的塑钢窗框上。
    - **8-12秒**：女生掀开半拉棉麻窗帘，脸贴在玻璃上，窗外悬浮着巴掌大的银灰色球形物体，表面闪烁淡绿色光点，女生瞪大双眼捂住嘴巴。
4.  **AI视频Prompt**：

真实实拍、非CG非动画，4K超高清，独居客厅暖黄落地灯氛围光，女生裹着珊瑚绒沙发毯追喜剧，凑近窗台掀开窗帘，窗外悬浮银灰色球形外星物体带着淡绿色光点，氛围从松弛转为惊悚，细节真实自然

## 剧本3

2.  **故事原型**：夜班店员在整理关东煮货架，冰柜传来沉闷异响，打开柜门发现原本整齐的饭团被搅乱，内壁附着诡异的节肢状外星物体，店员的手瞬间僵住。
3.  **12秒改编**：
    - **0-4秒**：便利店暖白色LED灯光铺满全屋，店员戴着藏青鸭舌帽低头整理关东煮锅，汤面冒着白色热气，货架上摆满袋装零食，收银台扫码枪亮着淡蓝色指示灯。

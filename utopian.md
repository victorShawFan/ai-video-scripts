# 乌托邦题材12秒视频剧本集

## 剧本1
1.  **改编自**：反乌托邦起义母题
2.  **故事原型**：被严密管控的穹顶城市中，底层居民终生未曾见过外界的阳光。一名维修工人意外发现穹顶外壳的隐秘缺口，窥见了穹顶外澄澈的蓝天与青绿草地。他决定打破虚假的安逸，将真相传递给身边的人。
3.  **12秒改编**：
    - **0-4秒**：狭小逼仄的地下维修间，昏黄的钨丝灯滋滋作响，沾满油污的工人蹲在控制台前，指尖触碰到闪烁红光的异常节点，眉头紧锁，鼻尖萦绕着机油和闷湿尘土的味道。
    - **4-8秒**：工人顺着爬梯抵达穹顶内侧，粗糙的合金外壳冰凉刺骨，他用扳手撬开松动的盖板，刺眼的自然光倾泻而下，照亮他满是错愕的侧脸。
    - **8-12秒**：工人攥着破损的盖板，冲着下方围观的同伴用力挥手，嘴角扬起释然的笑，光斑在他沾满油污的工装上来回晃动。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K超清，地下维修间昏黄钨丝灯光线，工人蹲在布满油污的控制台前，指尖触碰红色闪烁节点，眉头紧锁；切到穹顶内侧，工人撬开盖板，自然光倾泻照亮错愕侧脸；最后工人挥手微笑，光斑打在沾满油污的工装，画面温暖治愈
~~~

## 剧本2
1.  **改编自**：乌托邦治愈重逢母题
2.  **故事原型**：战后的废土之上，幸存者们搭建起自给自足的移动聚落。一名老人保留着战前留存的旧照片，一直在寻找照片里种植着大片向日葵的旧日家园。终于在迁徙途中，他找到了被遗忘的向日葵花田。
3.  **12秒改编**：
    - **0-4秒**：斑驳的铁皮移动房车旁，老人坐在折叠木凳上，指尖摩挲泛黄的旧照片，侧脸被落日暖光包裹，鼻尖萦绕着柴油和干燥枯草的气息。
    - **4-8秒**：老人拄着磨得发亮的木质拐杖，沿着碎石小路缓步前行，视线突然定格，瞳孔微微睁大，嘴角缓缓扬起柔和的弧度。
    - **8-12秒**：老人站在齐腰高的向日葵花田中，风拂动花盘沙沙作响，他将旧照片举到花田旁对比，眼角泛起细碎的湿润光泽。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K超清暖色调，老人坐在移动房车旁摩挲旧照片，侧脸被落日暖光包裹；切到老人拄拐前行，定格在向日葵花田前；最后老人举着旧照片对比花田，眼角带泪微笑，画面温馨治愈
~~~

## 剧本3
1.  **改编自**：乌托邦集体协作母题
2.  **故事原型**：漂浮在海面的水上聚落遭遇罕见的风暴预警，所有居民放下各自的工作，合力加固浮桥与房屋。原本存在隔阂的邻里们在共同的危机中放下矛盾，一同守护他们仅有的家园。
3.  **12秒改编**：
    - **0-4秒**：灰蓝色的暴雨将至，铅灰色云层压得极低，居民们攥着粗麻绳站在晃动的浮桥边缘，雨衣被海风掀动，鼻尖萦绕着咸腥的海水气味。
    - **4-8秒**：居民们两两一组合力拉扯麻绳，绷紧的绳索勒进掌心，所有人都抿着嘴唇眼神专注，雨水开始零星打落在他们的安全帽上。
    - **8-12秒**：最后一根木桩被钉入浮箱，居民们相视点头露出笑容，远处海面的雨幕中透出细碎的金色阳光，照亮沾着水珠的雨衣衣角。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K超清冷色调转暖色调，暴雨将至时居民们攥着粗麻绳站在浮桥边缘，雨衣被海风掀动；切到居民合力拉扯绳索，雨水打落在安全帽上；最后钉完木桩相视微笑，雨幕透出金阳光线，画面充满力量感
~~~

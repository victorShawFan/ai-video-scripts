## 剧本001：密室杀人（东野圭吾风格）

**改编自**：东野圭吾《嫌疑人X的献身》推理结构

**原作核心**：用数学逻辑构建完美不在场证明，献身式的爱情悲剧

**12秒改编**：
- **0-3秒**：警察勘察密室现场，门窗从内反锁，死者倒在书桌前
- **3-6秒**：侦探检查现场："死亡时间晚上8点，但嫌疑人有完美不在场证明"
- **6-9秒**：特写：窗台上融化的冰块，侦探拿起冰块残留的水痕
- **9-12秒**：侦探："凶手用冰块制作了反锁装置，杀人后让冰融化消除证据"

**AI视频Prompt**：
```
[豪华书房密室] 警察勘察反锁房间死者倒地，侦探特写检查窗台冰块水痕，慢镜头展示冰块融化过程，侦探眼神犀利指出真相，日式推理美学，冷色调光影，东野圭吾《嫌疑人X》科学推理风格
```

## 创作说明

**为什么基于这些IP？**
1. **东野圭吾**：科学推理+人性探讨，逻辑严密
2. **阿加莎·克里斯蒂**：黄金时代推理，结构经典
3. **名侦探柯南**：视觉化推理，适合短视频
4. **CSI/心理侦探**：现代科技+心理学，新颖

**12秒推理结构**：
- **0-3秒**：案件呈现（死亡+疑点）
- **3-6秒**：证据收集（侦探观察）
- **6-9秒**：推理过程（逻辑演绎）
- **9-12秒**：真相揭晓（破案瞬间）

**AI视觉化要点**：
- 强调"特写镜头"（证据细节）
- 使用"推理回放"（闪回+慢镜头）
- "分屏对比"（证据vs证词）
- 参考《名侦探柯南》推理演绎特效

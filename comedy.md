## 剧本001：扶不扶

**改编自**：开心麻花经典小品《扶不扶》（沈腾主演）

**故事原型**：好心人郝建骑车追汽车提醒后备箱没关，结果自己追尾被判全责。路遇老太太摔倒纠结要不要扶，最后用反向操作让老太太说出真相。

**核心笑点**："人倒了咱不扶，那人心不就倒了吗？"

**12秒改编**：
- **0-3秒**：男子骑车看到老太太摔倒，犹豫："扶不扶？上次扶完赔了两百块"
- **3-6秒**：老太太："哎呀我的腰啊！"男子装作什么都没看见往回走
- **6-9秒**：老太太追上来拽住他："你撞的我！"男子："大妈，我离你10米远"
- **9-12秒**：男子灵机一动自己躺地上："哎呀我的波棱盖儿！"老太太慌了："我没撞你啊"

**AI视频Prompt**：
```
[街道路边白天] 中年男子骑自行车路过看到老太太摔倒，犹豫表情特写，老太太追上拽住他衣服，男子突然自己躺地上装受伤，老太太慌张解释没撞人，反转幽默，开心麻花喜剧风格，明亮日光
```

## 剧本002：今天的幸福（未来儿子）

**改编自**：开心麻花小品《今天的幸福》

**故事原型**：丈夫邓小亮为让怀孕的妻子开心，找好友郝建假扮"从2042年穿越回来的未来儿子"，结果穿帮后妻子早已识破但很感动。

**核心笑点**：郝建从电视里"穿越"出来，身份证写着"邓眼珠子"

**12秒改编**：
- **0-3秒**：丈夫紧张地对妻子说："老婆，咱儿子要从未来穿越回来看你"
- **3-6秒**：电视屏幕突然破开，男子从里面爬出来："妈妈！我是你未来的孩子"
- **6-9秒**：妻子晕倒，醒来后"儿子"拿出身份证："邓眼珠子"，妻子："是我儿子没跑了"
- **9-12秒**：妻子："老公你为了哄我开心真是拼了"，抱住丈夫，"儿子"在旁边偷偷数钱

**AI视频Prompt**：
```
[现代客厅室内] 男子从电视屏幕破洞爬出喊妈妈，孕妇震惊后拿起身份证看到邓眼珠子三个字，夫妻拥抱感动，演员在旁边数钱，温馨搞笑反转，开心麻花风格，暖色家庭灯光
```

## 剧本003：办公室摸鱼

**改编自**：抖音博主"戏精张三"办公室情景剧

**故事原型**：员工上班摸鱼玩手机，老板突然出现在身后的经典桥段，三幕式反转结构。

- **9-12秒**：镜头切换到侧面，老板一直站在另一侧盯着他的手机屏幕看

**AI视频Prompt**：
```
[现代办公室隔断工位] 年轻员工偷看手机突然警觉收起装工作，确认是同事后继续玩，镜头切换到全景显示老板一直站在工位另一侧看着他手机屏幕面无表情，办公室喜剧，日光灯照明，《当代年轻人图鉴》风格
```

## 创作说明

**为什么这些剧本有效？**
1. **基于知名IP**：开心麻花是国民喜剧品牌，观众有认知基础
2. **真实共鸣**：办公室摸鱼、社恐、emo等都是当代年轻人真实处境
3. **反转节奏**：12秒内完成铺垫-冲突-反转三段式
4. **视觉化笑点**：表情、动作、场景对比都是可视化的喜剧元素

**AI生成建议**：
- 强调"表情特写"（夸张的面部表情是喜剧关键）
- 使用"反差对比"（期待vs现实、白天vs深夜）
- 参考风格：开心麻花舞台喜剧、抖音情景剧《当代年轻人图鉴》
- 光线：明亮温暖的喜剧光线，避免阴暗压抑

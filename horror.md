# 恐怖惊悚类剧本（12秒）

> **内容来源**：基于日本恐怖片经典IP改编，包括《午夜凶铃》《咒怨》《怪谈新耳袋·百物语》等知名作品的核心场景

## 剧本001：诅咒录像带

**改编自**：《午夜凶铃》（Ringu）- 贞子诅咒录像带桥段

**故事原型**：观看神秘录像带的人会在7天内死亡，女记者调查发现诅咒源自被推入井中的山村贞子

**12秒改编**：
- **0-3秒**：深夜客厅，女孩独自播放一盘老录像带
- **3-6秒**：屏幕闪现诡异画面：井、长发、扭曲的脸（快速闪切）
- **6-9秒**：电视突然黑屏，背后传来滴水声，女孩慢慢转头
- **9-12秒**：镜子里倒映出贞子从电视爬出的身影，女孩惊恐尖叫

**AI视频Prompt**：
```
[深夜昏暗的客厅] 年轻女孩坐在沙发上，老式电视机播放雪花屏，突然出现井底仰拍视角，长发遮面的贞子扭曲爬行，电视黑屏，女孩背后镜子反射贞子从屏幕爬出的恐怖画面，日式恐怖片风格，冷蓝色调，《午夜凶铃》经典氛围
```

**改编自**：《咒怨》（Ju-on）- 凶宅诅咒

**故事原型**：含怨而死的人在生前居住地形成诅咒，任何进入者都会遭遇不幸

**12秒改编**：

- **6-9秒**：抬头望去，天花板缓缓渗出黑色液体
- **9-12秒**：液体中浮现伽椰子扭曲的脸，客户吓瘫在地

**AI视频Prompt**：
```
[破旧日式住宅内部] 中介打开衣柜，天花板传来儿童诡异呻吟声，镜头仰拍天花板渗出黑色液体，液体中浮现长发女鬼伽椰子扭曲的脸，客户惊恐倒地，日本恐怖片压抑氛围，昏黄灯光，《咒怨》风格
```

## 剧本018：如月车站
  
**改编自**：日本都市传说《如月车站事件》- 2004年网络真实流传的灵异事件

**故事原型**：2004年一名日本女孩在网络论坛实时发帖称自己误入了地图上不存在的"如月车站"，手机无法定位，周围荒无人烟，最后失联。此事件在日本网络引发轰动，成为最著名的都市传说之一，许多人声称也遇到过类似诡异车站。

**12秒改编**：
- **0-3秒**：深夜电车上，女孩低头玩手机，抬头发现车厢只剩自己，窗外一片漆黑
- **3-6秒**：电车停靠，站牌显示"如月"，她打开手机地图发现定位失败，信号格消失
- **6-9秒**：走出车厢，月台空无一人，四周是荒废的站台和生锈的铁轨，远处传来诡异钟声
- **9-12秒**：回头发现电车已消失不见，月台尽头出现一个背对她的黑色人影，缓缓转身露出没有五官的脸

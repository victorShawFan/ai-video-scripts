## 剧本001：恐怖电梯之夜

**改编自**：《生活娱乐类短视频剧本-恐怖电梯之夜》道客巴巴案例
**角色**：小明（上班族）
**场景**：深夜办公楼电梯

**剧情**：
- 0-4秒：小明深夜加班后独自进入老旧电梯，按下1楼
- 5-8秒：电梯下降途中突然停在不存在的地下13层
- 9-12秒：门打开，漆黑走廊里有模糊人影在挥手，电梯门关上前看清是自己的脸

**视觉风格**：
- 镜头：电梯内固定机位+门口视角
- 光影：昏暗电梯灯+门外诡异绿光
- 色调：冷灰色调，高对比度

**完整Prompt**：
```
深夜办公楼，疲惫的年轻上班族小明走进空旷的老旧电梯，电梯灯光昏暗闪烁。他按下1楼按钮，电梯门缓缓关闭开始下降。经过3楼、2楼后，数字显示突然跳到"B13"，但这栋楼只有3层地下室。电梯停下，伴随诡异的"叮"声，门缓缓打开。门外不是停车场，而是一条漆黑的走廊，远处有模糊的人影在向他招手。小明想按关门键，但电梯失灵了。走廊里的人影慢慢走近，灯光照亮那张脸——居然是他自己！门在最后一刻关上。电梯内固定视角，绿色诡异光线，压抑音效。
```

## 剧本002：镜子里的诡异

**改编自**：B站恐怖短片《镜子》及Reddit nosleep经典故事
**角色**：女孩
**场景**：夜晚卫生间

**剧情**：
- 0-4秒：女孩深夜洗漱，对着镜子刷牙
- 5-8秒：镜中倒影动作开始滞后半拍
- 9-12秒：镜中的自己停止动作，直视镜外，露出不属于她的笑容

**视觉风格**：
- 镜头：镜内镜外对切
- 光影：卫生间白炽灯+镜面反光
- 色调：冷白色，镜面扭曲效果

**完整Prompt**：
```
凌晨的卫生间，柔和的白炽灯照亮洗手台。年轻女孩站在镜子前刷牙，镜中的倒影和她的动作完全同步。突然，她察觉到一丝违和——镜子里的自己，动作好像慢了零点几秒。她停下来盯着镜子，试探性地抬起左手，镜中的"她"也抬手，但明显有延迟。她开始不安，做了个挥手动作，镜中的"她"竟然没有跟随，而是静止不动，缓缓转过头，不再看向应该看的方向，而是直直地凝视镜子外的她，嘴角诡异上扬，露出一个冰冷的笑容，那不是她的表情。镜内镜外快速切换，微妙光线变化，略微扭曲色调。

## 剧本003：深夜敲门的真相

**改编自**：创意策划网《半夜敲门》及网络都市传说
**角色**：独居女性
**场景**：公寓

**剧情**：
- 0-4秒：女子听到敲门声，通过猫眼查看，走廊空无一人
- 5-8秒：转身回房，敲门声再次响起，更急促
- 9-12秒：意识到敲门声来自身后紧闭的卧室门内

**视觉风格**：
- 镜头：第一人称视角+猫眼畸变效果
- 光影：昏暗室内+门缝光线
- 色调：暗蓝色夜间氛围

**完整Prompt**：
```
深夜公寓，独居女子正在客厅看书，突然传来三声沉闷的敲门声"咚咚咚"。她警惕地放下书，走到门口透过猫眼向外看——走廊空荡荡的，只有昏黄的感应灯在闪烁。她松了口气，以为是幻听，转身准备回房间。刚走几步，敲门声再次响起，这次更加急促"咚咚咚咚"，像是有人在用力拍门。她僵住了，慢慢回头看向门口，但随即瞳孔放大——声音不是从入户门传来的，而是从她身后紧闭的卧室门内！她独自在家，卧室里不应该有人。第一人称视角，猫眼畸变效果，暗蓝色调，低频音效营造恐怖氛围。
```

```

## 剧本010：窗外25楼的手

**改编自**：高层住宅都市传说
**角色**：失眠者
**场景**：25楼卧室

**剧情**：
- 0-4秒：25楼卧室，窗外传来敲击声
- 5-8秒：以为是风，敲击变得有规律
- 9-12秒：起身查看，窗外有只苍白的手在敲窗

**视觉风格**：
- 镜头：从床到窗边移动
- 光影：月光洒进
- 色调：冷蓝月光

**完整Prompt**：
```
深夜25楼卧室，失眠者躺在床上辗转反侧，窗外月光洒进来。突然，窗户上传来细微的"嗒嗒"声，像有东西在敲击玻璃。他以为是风吹，翻身继续躺。但敲击声没有停止，反而更清晰、更有规律，"嗒嗒嗒"，一下接一下，像有人在敲门。他不安地坐起身，犹豫片刻后决定起床查看。走到窗边拉开窗帘的瞬间，瞳孔急剧放大——窗外玻璃上，贴着一只苍白瘦长的手，手指一下一下地敲着窗，手腕之上消失在黑暗中。这是25楼，窗外应该什么都没有。从床上视角移动到窗边，月光照射，冷蓝色调，敲击声由远及近，恐怖氛围音效。
```

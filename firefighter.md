# 消防救援题材12秒视频剧本集
## 剧本1
1.  **改编自**：深夜居家火情营救母题
2.  **故事原型**：老旧居民楼雨夜突发火情，独居老人被困浴室无法脱身，浓烟封住楼道逃生路线，消防员接警后第一时间奔赴现场展开营救。
3.  **12秒改编**：
    - **0-4秒**：雨夜的老旧红砖居民楼三楼浴室窗口冒出灰黑色浓烟，楼道声控灯忽明忽暗，消防员半蹲在门口，湿透的战斗服反光条在昏暗里格外醒目，抬手用破门锤砸向浴室木门。
    - **4-8秒**：木门被砸开后，消防员弯腰冲进满是烟雾的浴室，看见老人瘫在浴缸边缘裹着湿毛巾，立刻将呼吸面罩戴在老人头上。
    - **8-12秒**：消防员半扶半背着老人躬身冲出楼道，楼下消防应急灯在雨幕里闪烁暖光，两人身影融入楼下等候的救援队伍。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K分辨率，雨夜昏暗打光，老旧红砖居民楼楼道场景，消防员身穿带反光条的湿透战斗服，动作急促却沉稳，镜头跟随消防员视角推进，还原烟味弥漫的氛围感，细节还原消防救援真实流程
~~~

## 剧本2
1.  **改编自**：孩童遇险应急营救IP母题
2.  **故事原型**：小区地下车库电动车自燃，留守孩童躲在消防栓后方不敢挪动，浓烟在密闭车库快速扩散，巡逻消防员第一时间发现火情并定位被困孩童。
3.  **12秒改编**：
    - **0-4秒**：地下车库黄灯闪烁，空气中弥漫焦糊味，电动车残骸冒着零星火星，消防员跪在消防栓旁，轻轻拨开挡在孩童身前的塑料围挡。
    - **4-8秒**：消防员将防烟面罩扣在孩童脸上，孩童攥紧消防员沾着黑灰的手腕微微发抖，消防员轻拍孩童后背安抚情绪。
    - **8-12秒**：消防员抱着孩童快步跑出车库入口，正午阳光刺眼，远处消防车警笛声渐停，两人身影沐浴在室外阳光下。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K分辨率，地下车库昏暗压抑的光线，消防员身着整洁战斗服，动作轻柔却果断，捕捉孩童受惊的微表情，还原现场焦糊气味的氛围感，整体画面偏向纪实风格

## 剧本3
1.  **改编自**：高空极限营救母题
2.  **故事原型**：建筑工地高空脚手架突发倾斜，工人被卡在钢管缝隙当中脚下悬空摇摇欲坠，消防高空救援队携带速降装备赶赴现场展开救援。
3.  **12秒改编**：
    - **0-4秒**：正午烈日下，白色脚手架扭曲倾斜，工人半个身体卡在钢管缝隙里，速降绳索在浅蓝天空中绷紧，消防员抓着绳索快速下滑靠近工人。
    - **4-8秒**：消防员用便携切割器剪断卡住工人的变形钢管，工人手臂沾着红色液体，消防员拿出防水绷带快速缠绕固定。
    - **8-12秒**：消防员挂着安全绳带着工人缓缓降落到地面，地面队友立刻上前接应，两人瘫坐在安全垫上喘着气。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K分辨率，正午强光下的高空场景，消防员身穿专业高空救援服，动作精准利落，细节还原脚手架扭曲的金属质感，弱化红色液体的视觉冲击，整体画面偏向硬核纪实风格
~~~

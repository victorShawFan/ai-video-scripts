# AI视频生成剧本库

> 12秒短视频剧本Prompt集合，基于网络爆款案例改编

## 📖 项目简介

本仓库收集网络上真实流行的短视频创意，并改编成适合AI视频生成的剧本Prompt。每个剧本都经过精心设计，适合12秒短视频，支持1-3个角色，可直接用于Runway、Pika、可灵等AI工具。

**内容来源**：
- 抖音/TikTok爆款案例
- Reddit r/nosleep 经典故事
- B站高播放创意短片
- 办公室搞笑段子改编
- 真实生活场景提炼

## 🎬 剧本规格

| 属性 | 规格 |
|------|------|
| 视频时长 | 12秒 |
| 角色数量 | 1-3人 |
| 剧本结构 | 建立→冲突→反转 |
| Prompt长度 | 150-300字 |

## 📂 剧本分类

### 1. 恐怖惊悚类 (`horror.md`)
- 基于《午夜凶铃》《咒怨》《怪谈新耳袋》等日本恐怖IP
- 18个经典恐怖场景剧本
- 贞子、伽椰子、都市传说

### 2. 搞笑喜剧类 (`comedy.md`)
- 基于开心麻花小品、抖音《当代年轻人图鉴》IP
- 15个职场生活喜剧剧本
- 《扶不扶》《今天的幸福》《落叶归根》等经典

### 3. 悬疑推理类 (`mystery.md`)
- 基于东野圭吾、阿加莎·克里斯蒂、《名侦探柯南》IP
- 15个经典推理案件剧本
- 密室杀人、时间诡计、心理侧写

## 📝 Prompt格式

```markdown
### 剧本标题

**改编自**：来源说明
**角色**：1-3个角色
**场景**：主要场景

**剧情**：
0-4秒：开场
5-8秒：冲突
9-12秒：反转/高潮

**视觉风格**：镜头、光影、色调

**完整Prompt**：
[适合AI工具的描述]
```

## 🎯 使用方法

1. 选择喜欢的剧本
2. 复制完整Prompt
3. 输入到AI视频工具：
   - Runway Gen-3
   - Pika Labs  
   - 可灵AI
   - Luma Dream Machine
4. 根据效果调整参数

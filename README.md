# AI视频生成剧本库

> 12秒短视频剧本Prompt集合，覆盖50+种类，每类100+剧本

## 📖 项目简介

本仓库收集和整理各种类型的AI视频生成剧本Prompt，专为12秒短视频设计。每个剧本支持1-3个角色出场，结构紧凑，冲突明确，适合AI视频生成工具（如Runway、Pika、可灵等）使用。

## 🎬 剧本规格

| 属性 | 规格 |
|------|------|
| 视频时长 | 12秒 |
| 角色数量 | 1-3人 |
| 剧本结构 | 开场→冲突→反转/高潮 |
| Prompt长度 | 100-300字 |

## 📝 Prompt格式

每个剧本包含以下元素：

```markdown
### 剧本标题

**类型**：子类型标签
**角色**：角色1、角色2、角色3
**场景**：主要场景描述

**剧情**：
开场（0-4秒）：场景建立，角色出场
发展（5-8秒）：冲突产生，情节推进
高潮（9-12秒）：反转或高潮，强烈情绪

**视觉风格**：镜头、光影、色调
**音效提示**：背景音乐、音效建议

**完整Prompt**：
[适合直接输入AI视频工具的完整描述]
```

## 🎯 使用方法

### 1. 选择剧本
浏览对应类别的md文件，找到喜欢的剧本

### 2. 复制Prompt
复制"完整Prompt"部分

### 3. 输入AI工具
- **Runway Gen-3**: 直接输入prompt
- **Pika Labs**: 可添加运动参数
- **可灵AI**: 支持中文prompt
- **Luma Dream Machine**: 简洁prompt效果更好

### 4. 调整参数
根据实际效果调整：
- 镜头运动（推拉摇移）
- 时长控制（8-15秒）
- 画面风格（写实/动画/复古）

# 偶像题材12秒视频剧本集
## 剧本1
1.  **改编自**：校园暗恋双向奔赴母题
2.  **故事原型**：怯懦的普通女生一直悄悄关注着学校里耀眼的偶像学长，不敢上前搭话。学长其实也留意到了这个总在远处偷看自己的女孩，默默记住了她常带的草莓发绳。两人在图书馆转角意外偶遇，终于打破了距离。
3.  **12秒改编**：
    - **0-4秒**：暖黄色午后阳光斜洒在图书馆浅棕色木质书桌，女生攥着印着草莓图案的米白色帆布包，指尖捻着藏在袖口里的草莓发绳，偷瞄斜对面抱着原木色吉他翻看乐谱的学长，耳尖泛红，连呼吸都放轻，桌角放着半杯冒着热气的柠檬水。
    - **4-8秒**：学长抬手拨了额前浅棕色碎发，余光瞥见女生，悄悄将自己笔袋上挂着的同款草莓发绳露出来，假装翻书时侧头朝女生方向弯了弯眼，眼尾带着细碎笑意。
    - **8-12秒**：女生抬头撞上学长的目光，先是惊慌低头，随即看到他笔袋上的发绳，猛地睁大眼睛，嘴角不受控制地扬起，轻轻抬手挥了挥。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K超清，暖黄色午后自然光，校园图书馆场景，木质书桌带着浅淡木纹，女生耳尖泛红羞涩偷瞄的神态细节还原，学长温柔弯眼露出同款草莓发绳，镜头捕捉帆布包和笔袋的草莓元素，整体氛围清甜治愈。
~~~

## 剧本2
1.  **改编自**：舞台救赎双向治愈母题
2.  **故事原型**：曾经闪耀的偶像因为舆论跌入低谷，躲在老剧场不肯再登台。年少时被他的舞台照亮过的小粉丝成长为剧场助理，悄悄留下了当年他的应援棒，终于让他重新拿起话筒。
3.  **12秒改编**：
    - **0-4秒**：昏暗的老旧剧场后台，积灰的铜框落地镜前，男人垂着肩盯着手里磨旧的黑色话筒套，暖黄台灯光只照亮他半边侧脸，下颌线紧绷，指节微微泛白，镜面上还贴着半张泛黄的旧海报残片。
    - **4-8秒**：他转身时瞥见化妆台上放着一根蒙着薄灰却还亮着暖橙色灯光的应援棒，贴纸已经起皱边角翻卷，正是当年他首场演出的限定款式。
    - **8-12秒**：男人伸手握住应援棒，指尖拂过起皱的贴纸，慢慢抬起头看向镜中的自己，原本黯淡的眼神重新亮起，抬手将话筒举到嘴边，指节轻轻搭在话筒上。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K超清，老旧剧场后台场景，暖黄台灯光调柔和朦胧，还原男人垂肩落寞的神态细节，应援棒暖橙色灯光点亮昏暗角落，画面充满复古质感，整体氛围带着治愈的拉扯感。
~~~

## 剧本3
1.  **改编自**：反差偶像私下温柔日常母题
2.  **故事原型**：镜头前永远活力满分的顶流偶像，私下里却总在结束行程后独自去巷口喂流浪猫。刚入行的新人助理偶然发现了这一幕，才知道耀眼的偶像也藏着不为人知的温柔一面。
3.  **12秒改编**：
    - **0-4秒**：深夜巷口的暖橙色路灯下，地面飘着烤红薯的甜香，偶像脱下了舞台上的亮片外套，裹着宽松的黑色连帽衫，帽檐半压着眉眼，蹲在墙边，指尖轻轻挠着三花流浪猫的下巴，眉眼柔和得不像镜头前的样子，脚边放着半袋拆开的猫粮。
    - **4-8秒**：躲在巷口拐角的新人助理攥着打包的热奶茶，悄悄探出头，看到这一幕后惊讶地捂住嘴，眼底满是动容，奶茶杯上的热气模糊了她的半张脸。
    - **8-12秒**：偶像听到动静回头，看到助理后没有慌乱，反而扯下连帽衫帽子，露出带着细碎汗珠的额头，笑着朝她招了招手，指了指脚边的空猫碗示意她过来。
4.  **AI视频Prompt**：
~~~
真实实拍、非CG非动画，4K超清，深夜暖橙色路灯光影，巷口飘着烤红薯甜香，还原偶像蹲下身温柔撸猫的细节，助理捂嘴惊讶的神态自然生动，画面氛围温暖治愈，充满生活真实感。
~~~

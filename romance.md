## 剧本001：眼泪test

**改编自**：第一抖音剧本网《情侣约会段子》

**角色**：
- 女生（可爱，试探男友）
- 男生（直男，但有撩人瞬间）

**场景**：咖啡厅，柔和灯光

**剧情**：
- **0-3秒**：女生："你们说如果眼泪流下来，会怎么样？"
- **3-6秒**：男生想了想："胸大滴胸上，胸小滴脚上吧"
- **6-9秒**：女生羞涩："那你看我…"
- **9-12秒**：男生温柔笑："你肯定不会流眼泪，因为有我在"（女生破防）

**视觉风格**：
- 暖色调，咖啡厅氛围灯
- 特写：女生眼睛闪烁、男生温柔表情
- 背景音：轻柔钢琴曲

**完整Prompt**：
```
[温馨咖啡厅] 可爱女生试探性提问，直男男友先是直男回答惹笑，最后突然浪漫转折"因为有我在"，女生感动，暖色调柔光，甜蜜氛围，韩剧风格
```

## 剧本002：彩礼反转

**改编自**：道客巴巴《搞笑段子剧本大全·彩礼》

**角色**：
- 新娘（美丽，坚定）
- 新郎（紧张）
- 朋友（旁观）

**场景**：婚礼现场，喜庆布置

**剧情**：
- **0-3秒**：婚礼现场，新娘突然停住："等等，我有话要说"
- **3-6秒**：众人紧张，新娘严肃："彩礼问题我必须说清楚"
- **6-9秒**：新郎额头冒汗，新娘转头微笑："谢谢你没要我家一分彩礼"
- **9-12秒**：新郎感动拥抱，全场欢呼鼓掌（反转感动）

**视觉风格**：
- 喜庆红色主色调
- 分镜：新娘特写→众人反应→新郎表情→拥抱全景
- 背景音：先紧张后欢快的婚礼进行曲

**完整Prompt**：
```
[婚礼现场] 喜庆布景，新娘突然严肃停止仪式谈彩礼，镜头特写众人紧张表情，新娘转折感谢新郎没要彩礼，两人拥抱，全场欢呼，温馨感动，电影级婚礼摄影
```

## 剧本003：年龄焦虑与爱情

**改编自**：原创力文档《8个短视频剧本_年龄段子》

**角色**：
- 男子（30岁左右，豁达）

**场景**：居家卧室，男子对镜头

**剧情**：
- **0-3秒**：男子调侃状："朋友们，现在的年龄你们着不着急？"
- **3-6秒**："白素贞1000多岁下山谈恋爱，佘太君百岁挂帅，姜子牙80岁成为丞相"
- **6-9秒**："所以啊，你才30岁着什么急？"
- **9-12秒**：（画外音女友）："那你什么时候娶我？"男子：（尴尬）"...明天！"

**视觉风格**：
- 生活化场景，自然光
- 男子表情从自信到尴尬的转变
- 背景音：轻快bgm突然停顿

**完整Prompt**：
```
[居家场景] 男子对镜头调侃年龄焦虑引用历史人物段子，表情自信轻松，突然女友画外音质问，男子表情秒变尴尬求生欲，生活vlog风格，真实自然
```

## 剧本004：闻见恋爱的酸臭味

**改编自**：原创力文档《精选抖音短视频恋爱情感类·闻见了恋爱的酸臭味》

**角色**：
- 女主（暗恋男神）
- 男神（帅气，迟钝）

**场景**：街边夜景，霓虹灯

**剧情**：
- **0-3秒**：女主和男神逛街，女主内心独白："今天一定要表白"
- **3-6秒**：女主鼓起勇气："你有没有闻到什么味道？"
- **6-9秒**：男神认真嗅："没有啊...什么味道？"
- **9-12秒**：女主脸红："恋爱的酸臭味...我喜欢你"（男神愣住后笑了）

**视觉风格**：
- 城市夜景霓虹灯背景
- 特写：女主紧张表情、男神从疑惑到惊喜
- 背景音：心跳声+浪漫旋律

**完整Prompt**：
```
[城市街道夜景] 霓虹灯闪烁，女生鼓起勇气表白"闻到恋爱酸臭味"，男生从疑惑到恍然大悟到惊喜的表情变化，浪漫氛围，日系青春片风格
```

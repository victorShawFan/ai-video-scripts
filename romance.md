## 剧本001：星巴克遇见你

**改编自**：抖音博主"奶茶和咖啡"日常系列

**核心甜点**：陌生人咖啡杯写错名字的浪漫邂逅

**12秒改编**：
- **0-3秒**：女生在星巴克等咖啡，听到店员喊"小美的拿铁"
- **3-6秒**：她拿起杯子发现不是自己的名字，转身看到帅气男生也拿错了
- **6-9秒**：两人对视笑了，交换杯子时男生："缘分让我们喝错了彼此的咖啡"
- **9-12秒**：女生看杯子上男生留的电话号码，脸红微笑

**AI视频Prompt**：
```
[星巴克店内暖色调] 女生听到名字拿咖啡发现杯子写错，与帅气男生同时意识到拿错，交换杯子时四目相对微笑，特写杯子上偷偷写的电话号码，都市爱情，咖啡店柔和灯光，抖音甜宠风格
```

## 创作说明

**为什么这些剧本有效？**
1. **真实共鸣**：基于真实恋爱场景，观众能代入
2. **甜度适中**：不狗血不夸张，日常小甜蜜
3. **视觉化情绪**：脸红、对视、心跳都能通过镜头表现
4. **现代感**：奶茶、朋友圈、外卖等当代元素

**12秒甜宠结构**：
- **0-3秒**：日常场景设定
- **3-6秒**：心动触发点
- **6-9秒**：情绪升温
- **9-12秒**：甜蜜结局

**AI视觉化要点**：
- 强调"特写"（脸红、对视、牵手）
- 使用"柔光"（营造浪漫氛围）
- "暖色调"（增加温馨感）
- 参考日韩甜宠剧美学
